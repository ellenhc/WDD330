Chp 2: Programming Basics

Variables

Variables are used in programming languages to refer to a value stored in memory.

Variables have to be declared before they can be used. This is done with "const" or "let."

Scope

Scope refers to where a contstant or variable is accessible by the program.

There are two common scopes: global and local. Global scope means it is accessible everywhere in the program. For example, a variable declared outside of a block is said to have global scope. By contrast, blocks can be used to create a local scope. Variables defined insde a block will only be available inside that block.

Template Literals

Template literals are a special type of string that use the backtick character ` to deliminate the string.

This allows for interpolation of JS code, meaning that a JS expression can be inserted inside a string and the result will be displayed. The JS expression is placed inside the curly braces with a $ character in front.

Template literals can be thought of as super-powered strings since they behave the same way as normal strings, but they have the extra power of interpolation.

Converting/Parsing

Converting Strings to Numbers

The best way to change a string to a number is to use the Number method. If the string cannot be converted into a number, then NaN is returned.

Converting Numbers to Strings

The preferred way of changing a number to a string is to use the String function.

Another option is to use the toString() method.

Parsing Numbers

The function parseInt() can be used to convert a string representation of a numerical value back into a number.

  1. If a string starts with a number, parseInt will use the number and ignore any letters that follow.
  2. It will remove anything after the decimal point. This is not rounding, just removing.

parseFloat() converts strings into floating point decimal numbers.

Array Methods

Length Property

  1. Can be used to find the last item in an array
  2. It's mutable, meaning you can manually change it

Pop, Push, Shift, and Unshift

Pop removes the last item from an array and returns the last item.

Shift removes the first item in the array.

Push adds a new value to the end of the array.

Unshift adds a new item to the beginning of the array.

Merging Arrays

The concat() method can be used to merge an array with one or more arrays. It doesn't change the original array, but rather creates a new array with combined arrays.

The join() Method

Can be used to turn the array into a string that comprises all the items in the array as a list.

While the default is to separate items with a comma, you can choose a different separator by placing it inside the parentheses.

Slicing and Splicing

The slice() method creates a subarray. It's like chopping out a slice of the original array. It's non-destructive.

The splice() method removes items from an array and then inserts new items in their place. This is destructive because it permanently changes the array. It can also be used to insert values into an array at a specific index without removing any items if you indicate that zero items are to be removed.
